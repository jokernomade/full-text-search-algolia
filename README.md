# full-text-search-algolia
Full text search com Algolia em 1 minuto. Crie uma conta no Algolia e substitua as chaves ou utilize as chaves desse projeto enquanto estiverem disponíveis.


# executar servidor
`yarn server`

# executar reactjs
`yarn start`
